## 1. Reddit Post Draft

**Recommended Subreddits**: `r/productivity`, `r/opensource`, `r/selfhosted`, `r/electronjs`

**Post Title**:
> Show r/productivity: AuraTranslate – A sleek, command-palette style desktop translator that stays hidden until you need it

**Post Body**:
```markdown
Hey everyone,

I wanted a translation utility that feels as fast and clean as Spotlight or Raycast. Many existing desktop clients are bulky, slow, or constantly open in the taskbar. 

So I built **AuraTranslate** – a lightweight, borderless desktop command-palette translator that stays out of your way.

### Core Features:
* ⌨️ **Hotkey Activated**: Toggle it instantly with `Ctrl+Shift+Space` (or customize your own).
* 🖱️ **Auto-Hide (Hover Footer)**: The status footer collapses into a transparent 10px strip at the bottom of the window. Hovering over it expands it to show settings and active hotkeys.
* 🌓 **Frosted Glass Styling**: Supports gorgeous Dark, Light, and macOS-style Frosted Glass themes.
* 🔌 **Multiple Engines**: Works out-of-the-box with built-in free Google Translate, custom Google Apps Script URLs, or Gemini API keys.
* ⚡ **Ultra Fast**: Standalone portable build is only 92MB and launches instantly.

It's completely free and open-source!

* 📁 **GitHub Repository**: https://github.com/Crimson5775/AuraTranslate
* 🚀 **Download link**: https://github.com/Crimson5775/AuraTranslate/releases/download/v1.0.0/AuraTranslate.exe

Let me know what you think or if you have any feature suggestions!
```

**Post Body**:
```markdown
I built AuraTranslate because I wanted a translation tool that behaved like a native command palette (Raycast/Spotlight). 

It has a borderless, translucent interface, supports custom hotkeys, and defaults to a hover-only footer so the layout stays completely clean.

It includes keyless Google Translate, custom Google Apps Script, and Gemini API backends. Built using Electron, React, and Vite.
```
